# 06. Manual d'Usuari

## Requisits del Sistema
- **Python**: Versió 3.7 o superior.
- **Biblioteca Pygame**: Es pot instal·lar amb `pip install pygame`.

## Com Jugar
1. Executa el fitxer principal: `python src/mini_street_fighter.py`.
2. Al menú principal, selecciona el personatge utilitzant les tecles de direcció.
3. Prem `ENTER` per començar el combat.

## Controls

| Acció | Tecla |
|-------|-------|
| Moure a l'esquerra | `A` |
| Moure a la dreta | `D` |
| Saltar | `W` |
| Atacar (Puny/Patada) | `ESPACIO` |
| Atac Especial | `S` + `ESPACIO` |
| Sortir del joc | `ESC` |

## Consells de Joc
- Utilitza el salt per esquivar els projectils de l'enemic.
- El personatge **Zangief** té molta vida però és lent; utilitza'l si prefereixes un estil defensiu.
- **Chun-Li** és la més ràpida, ideal per atacar i retrocedir ràpidament.
