# 05. Millores Realitzades i Futures

## Millores Implementades
Durant el desenvolupament, s'han afegit les següents funcionalitats no previstes inicialment:
- **Sistema de Partícules**: Efectes visuals quan un personatge rep un cop.
- **Screen Shake**: Tremolor de pantalla en rebre atacs potents per augmentar el "feedback".
- **Fons Dinàmic**: Ciutat nocturna amb llums de neó que parpellegen.
- **Audio Sintètic**: Generació de sons mitjançant codi per evitar dependències de fitxers externs.

## Millores Futures
1. **Sistema de Combos**: Permetre encadenar atacs per fer més dany.
2. **Sprites Reals**: Substituir els rectangles per animacions de personatges reals.
3. **Nous Escenaris**: Diferents fons amb obstacles que afectin la jugabilitat.
4. **Mode Història**: Una sèrie de combats contra diferents enemics amb diàlegs.
